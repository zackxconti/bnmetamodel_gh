Welcome to bnmetamodel's documentation!
=======================================

.. note ::

   This package is currently in active development and using it should be done with caution.

   You are welcome to contribute to the development.


bnmetamodel is a python library, to build Bayesian Network metamodels from csv data. The library makes use of 'libpgm', which is a python library written by [authors] for building bayesian networks and performing inference.
