bnmetamodel_gh.BayesianNetwork module

class bnmetamodel_gh.BayesianNetwork.BayesianNetwork(BNdata: BNdata | None = None, netStructure: dict | None = None, modeldata: dict | None = None, targetlist: List | None = None, binranges: Any | None = None, verbose: bool | None = False)

A class used to represent a Bayesian Network.

Parameters:

  • BNdata (BNdata, optional) – The BNdata object containing the data to be used to build the Bayesian Network, by default None.

  • netStructure (dict, optional) – The structure of the Bayesian Network. A dictionary where keys are names of children and values are list of parent names.

  • modeldata (dict, optional) – The model data of the Bayesian Network, by default None.

  • targetlist (list, optional) – The list of targets of the Bayesian Network, by default None.

  • binranges (_type_, optional) – The bin ranges of the Bayesian Network, by default None.

  • verbose (bool, optional) – Whether to print the progress of the learning process, by default False.

convertEvidence(humanEvidence)

Converts human evidence to libpgm evidence.

Parameters:

humanEvidence (dict) –

Human evidence in the form of a dictionary, structured either as ranges of interest or hard numbers.

Example of a range of interest (min, max values in a list):

{
    "v1": [10, 20],
    "v2": [20, 40]
}

Example of hard numbers as values:

{
    "v1": [10],
    "v2": [30]
}
